# Навык Cartography

Cartography - это **кастомный навык**, включенный в этот репозиторий.

Он помогает агентам быстро сформировать качественную ментальную модель незнакомой кодовой базы, генерируя структурированный *codemap* и отслеживая изменения со временем.

## Что делает навык

Cartography предназначен для понимания репозитория и генерации иерархических codemap:

1. Выбирает релевантные файлы кода/конфигов с помощью LLM
2. Создает `.slim/cartography.json` для отслеживания изменений
3. Генерирует шаблоны `codemap.md` (по папкам), которые заполняют explorer-агенты

## Как использовать

Cartography устанавливается автоматически установщиком `oh-my-opencode-slim`, когда включены кастомные навыки.

### Запуск (вручную / локально)

Из корня репозитория (или с явным `--root`):

```bash
# Инициализация карты
python3 cartographer.py init --root /repo --include "src/**/*.ts" --exclude "node_modules/**"

# Проверка изменений
python3 cartographer.py changes --root /repo

# Обновление хэшей
python3 cartographer.py update --root /repo
```

## Результаты

### `.slim/cartography.json`

Файл отслеживания изменений с хэшами файлов/папок.

### `codemap.md` (в каждой папке)

Пустые шаблоны, создаваемые в каждой папке, чтобы агент в стиле Explorer мог заполнить:

- Зону ответственности
- Паттерны проектирования
- Поток данных/управления
- Точки интеграции
